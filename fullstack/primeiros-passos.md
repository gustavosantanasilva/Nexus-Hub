# 🔄 Primeiros Passos em Full Stack

Um desenvolvedor **Full Stack** trabalha tanto com **Frontend quanto Backend**.

Isso significa que ele pode criar **um site completo sozinho**.

---

## 🧑‍💻 O que um full stack faz

- cria a parte visual do site
- cria o sistema que roda por trás
- conecta o site ao banco de dados
- cria aplicações completas

---

## 🛠 Tecnologias usadas

| Área           | Tecnologias           |
| -------------- | --------------------- |
| Frontend       | HTML, CSS, JavaScript |
| Backend        | Node.js, Python       |
| Banco de dados | MySQL, MongoDB        |
| Versionamento  | Git                   |

---

## 🚀 Como começar

1. aprender frontend
2. aprender backend
3. aprender banco de dados
4. criar aplicações completas

---

## 💡 Projetos simples

- blog completo
- sistema de login
- site com cadastro de usuários
