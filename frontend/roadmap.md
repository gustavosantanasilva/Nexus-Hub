# 🌐 Roadmap de Estudos — Frontend

Este guia mostra **uma ordem simples de estudos** para quem quer aprender a criar sites e interfaces web.

---

# 🟢 Etapa 1 — Fundamentos da Web

Aprenda como os sites funcionam.

Estude:

- como funciona a internet
- o que é navegador
- como funciona um site

---

# 🟢 Etapa 2 — HTML

HTML é usado para **criar a estrutura das páginas**.

Aprenda:

- títulos
- parágrafos
- imagens
- links
- listas
- formulários

---

# 🟢 Etapa 3 — CSS

CSS é usado para **estilizar as páginas**.

Aprenda:

- cores
- fontes
- espaçamento
- layout
- responsividade

---

# 🟢 Etapa 4 — JavaScript

JavaScript faz o site **ganhar interatividade**.

Aprenda:

- variáveis
- condições
- loops
- funções
- manipulação da página

---

# 🟢 Etapa 5 — Git e GitHub

Aprenda a **salvar e compartilhar seu código**.

Aprenda:

- git init
- git commit
- git push
- GitHub

---

# 🟢 Etapa 6 — Projetos

Crie projetos simples:

- página pessoal
- portfólio
- lista de tarefas
- clone de site simples
