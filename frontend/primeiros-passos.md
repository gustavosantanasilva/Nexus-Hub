# 🌐 Primeiros Passos em Frontend

Se você quer aprender a **criar sites e interfaces de aplicativos**, a área de **Frontend** é um ótimo começo.

Frontend é a parte do site que **as pessoas veem e usam**, como:

- botões
- menus
- cores
- textos
- imagens
- animações

Quando você abre um site no navegador, tudo que aparece na tela foi feito por um **desenvolvedor frontend**.

---

# 🧑‍💻 O que um desenvolvedor frontend faz

Um profissional de frontend trabalha criando e melhorando a aparência de sites.

Algumas tarefas comuns são:

- criar páginas de sites
- organizar textos e imagens
- criar menus e botões
- fazer o site funcionar no celular e no computador
- adicionar interações (cliques, animações, etc)

---

# 🛠 Tecnologias que você vai aprender

Essas são as principais tecnologias usadas no frontend.

| Tecnologia     | Para que serve                    |
| -------------- | --------------------------------- |
| **HTML**       | criar a estrutura do site         |
| **CSS**        | definir cores, layout e aparência |
| **JavaScript** | adicionar interatividade          |
| **Git**        | salvar versões do código          |
| **GitHub**     | guardar projetos na internet      |

Não se preocupe se esses nomes parecem difíceis agora.
Você vai aprender **um por vez**.

---

# 🚀 Como começar

Uma boa ordem de estudo é:

1️⃣ Aprender **HTML**
2️⃣ Aprender **CSS**
3️⃣ Aprender **JavaScript**
4️⃣ Criar pequenos projetos
5️⃣ Aprender **Git e GitHub**

Com isso você já consegue criar **sites completos**.

---

# 💡 Projetos simples para iniciantes

Depois de aprender o básico, tente criar:

- uma página de perfil
- um site simples sobre você
- uma lista de tarefas
- um site de notícias simples

Esses projetos ajudam muito a **praticar e aprender mais rápido**.
