# 📱 Primeiros Passos em Desenvolvimento Android

Android é o sistema usado pela maioria dos **celulares do mundo**.

Um desenvolvedor Android cria **aplicativos para celulares**.

---

## 🧑‍💻 O que um desenvolvedor Android faz

- criar aplicativos
- melhorar a interface do app
- conectar o app a servidores
- publicar apps na Play Store

---

## 🛠 Tecnologias usadas

| Tecnologia     | Para que serve                 |
| -------------- | ------------------------------ |
| Kotlin         | linguagem principal            |
| Java           | linguagem tradicional          |
| Android Studio | programa para desenvolver apps |
| Firebase       | backend para apps              |

---

## 🚀 Como começar

1. aprender Kotlin
2. aprender Android Studio
3. criar aplicativos simples
4. publicar apps

---

## 💡 Projetos simples

- app de lista de tarefas
- app de notas
- app de lembretes
