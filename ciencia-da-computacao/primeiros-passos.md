# 🧠 Primeiros Passos em Ciência da Computação

Ciência da Computação ensina **como os computadores funcionam**.

É a base teórica usada por programadores.

---

## 📚 O que você aprende

- lógica de programação
- algoritmos
- estruturas de dados
- redes
- sistemas operacionais

---

## 🧑‍💻 Para que serve

Esses conhecimentos ajudam a:

- programar melhor
- resolver problemas
- entender como os sistemas funcionam

---

## 🚀 Como começar

1. aprender lógica de programação
2. estudar algoritmos
3. estudar estruturas de dados
4. resolver exercícios

---

## 💡 Atividades para iniciantes

- resolver problemas de lógica
- criar pequenos programas
- estudar funcionamento dos computadores
