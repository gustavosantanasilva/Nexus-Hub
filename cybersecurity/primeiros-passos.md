# 🔐 Primeiros Passos em Cybersecurity

Cybersecurity é a área que protege **computadores, sistemas e dados contra ataques**.

Hoje quase tudo está na internet, então **segurança digital é muito importante**.

---

## 🧑‍💻 O que um profissional de segurança faz

- encontrar falhas em sistemas
- proteger dados
- monitorar ataques
- testar segurança de sites

---

## 🛠 Ferramentas usadas

| Ferramenta | Para que serve             |
| ---------- | -------------------------- |
| Linux      | sistema usado em segurança |
| Wireshark  | analisar redes             |
| Burp Suite | testar sites               |
| Kali Linux | ferramentas de segurança   |

---

## 🚀 Como começar

1. aprender redes de computadores
2. aprender Linux
3. estudar segurança web
4. praticar em laboratórios

---

## 💡 Projetos simples

- estudar segurança de redes
- analisar tráfego de rede
- aprender testes de segurança
