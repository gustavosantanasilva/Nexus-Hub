# ☁️ Primeiros Passos em DevOps

DevOps é a área responsável por **colocar sistemas no ar e manter tudo funcionando**.

Eles ajudam os desenvolvedores a **publicar sites e aplicações na internet**.

---

## 🧑‍💻 O que um profissional DevOps faz

- publicar aplicações
- cuidar de servidores
- automatizar processos
- monitorar sistemas

---

## 🛠 Ferramentas usadas

| Ferramenta | Para que serve              |
| ---------- | --------------------------- |
| Linux      | sistema usado em servidores |
| Docker     | rodar aplicações            |
| Git        | controle de código          |
| CI/CD      | automatizar deploy          |

---

## 🚀 Como começar

1. aprender Linux
2. aprender Git
3. aprender Docker
4. aprender deploy de aplicações

---

## 💡 Projetos simples

- publicar um site
- criar um servidor simples
- automatizar deploy
