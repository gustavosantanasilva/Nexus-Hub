# ⚙️ Primeiros Passos em Backend

Backend é a parte da programação que **faz o sistema funcionar por trás do site**.

O usuário normalmente **não vê o backend**, mas ele é responsável por coisas como:

- salvar dados
- criar contas de usuário
- processar informações
- conectar o site ao banco de dados

---

## 🧑‍💻 O que um desenvolvedor backend faz

Algumas tarefas:

- criar sistemas e APIs
- salvar dados no banco
- criar sistemas de login
- fazer o site funcionar corretamente

---

## 🛠 Tecnologias usadas

| Tecnologia | Para que serve   |
| ---------- | ---------------- |
| Node.js    | criar servidores |
| Python     | criar sistemas   |
| Java       | sistemas grandes |
| SQL        | banco de dados   |
| Git        | salvar código    |

---

## 🚀 Como começar

1. aprender lógica de programação
2. escolher uma linguagem (ex: Python ou JavaScript)
3. aprender banco de dados
4. criar APIs simples

---

## 💡 Projetos simples

- sistema de cadastro de usuários
- API de lista de tarefas
- sistema de login
