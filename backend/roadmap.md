# ⚙️ Roadmap de Estudos — Backend

Backend é responsável pelo **funcionamento interno dos sistemas**.

---

# 🟢 Etapa 1 — Lógica de Programação

Aprenda:

- variáveis
- condições
- loops
- funções

---

# 🟢 Etapa 2 — Escolher uma Linguagem

Boas opções:

- JavaScript (Node.js)
- Python
- Java

---

# 🟢 Etapa 3 — Banco de Dados

Aprenda:

- o que é banco de dados
- como salvar dados
- consultas simples

Exemplo:

- MySQL
- PostgreSQL
- MongoDB

---

# 🟢 Etapa 4 — APIs

Aprenda a criar **APIs**, que permitem comunicação entre sistemas.

Aprenda:

- rotas
- requisições
- respostas

---

# 🟢 Etapa 5 — Autenticação

Aprenda:

- login
- cadastro
- proteção de rotas

---

# 🟢 Etapa 6 — Projetos

Crie projetos como:

- API de tarefas
- sistema de usuários
- sistema de login
